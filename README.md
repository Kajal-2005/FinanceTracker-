💰 Personal Finance Tracker
Track your income & expenses in Indian Rupees ₹

🚀 How to Run
Clone the repo
Open index.html in your browser
That's it — no install needed!

✨ Features
Add income & expenses in ₹
Live balance, income, expense cards
Spending breakdown by category
Data saves after browser refresh
Mobile friendly

🛠️ Tech Used
HTML, CSS, JavaScript
React 18 (CDN)
Angular-inspired Service pattern
Babel (runs JSX in browser)

📁 Structure
finance-tracker/
└── index.html  ← entire app in one file
Built for beginners — no terminal, no npm, just open and use.
